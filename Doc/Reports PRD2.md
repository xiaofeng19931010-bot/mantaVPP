## 1. 产品概述 (Product Overview)

### 1.1 产品简介

Reports 模块用于统一呈现 VPP 与 DER 两个层级的事件执行结果，支持实时与历史视角切换、状态与多维筛选、关键词检索、事件详情下钻和结果导出。该模块帮助运营团队完成事件复盘、执行追踪与收益核对，提升调度闭环效率与可解释性。

### 1.2 目标用户

- **VPP 运营经理 (VPP Operations Managers)**: 关注 VPP 级事件执行状态与整体收益表现。
- **调度与运维人员 (Dispatch & O\&M Operators)**: 关注 DER 级设备执行明细与失败原因定位。
- **数据与收益分析师 (Revenue Analysts)**: 关注事件收益、均价与历史区间导出分析。

### 1.3 核心价值

- **双层级洞察**: 同时覆盖 VPP 聚合视图与 DER 设备视图，支持从汇总到明细的快速下钻。
- **高效筛选复盘**: 通过状态、区域、事件类型、时间区间与搜索实现快速定位问题事件。
- **可导出留痕**: 支持按当前筛选条件导出结果，便于审计、汇报与离线分析。
- **统一口径展示**: 对状态与价格指标进行标准化展示，降低跨角色理解成本。

## 2. 交互与流程 (User Flow & Navigation)

### 2.1 站点地图 (Site Map)

- **一级菜单**: Reports (报表)
  - **二级菜单 A**: VPP Events
    - **页面 A1**: VPP Events 列表页 - *默认落地页*
    - **页面 A2**: VPP Event Details - *点击列表行 View Details 进入*
  - **二级菜单 B**: DER Events
    - **页面 B1**: DER Events 列表页

### 2.2 用户路径

1. 用户点击侧边栏 "Reports" -> "VPP Events"。
2. 系统默认展示 VPP 事件列表，用户可进行实时/历史切换、状态筛选、表头过滤与搜索。
3. 用户点击某条记录的 View Details，进入 VPP 事件详情页查看关联 DER 执行明细。
4. 用户可返回列表继续筛选，或切换至 "DER Events" 查看设备级事件执行结果并导出。

## 4. 非功能需求 (Non-Functional Requirements)

| 类别        | 需求描述                                | 验收标准                                                                         |
| :-------- | :---------------------------------- | :--------------------------------------------------------------------------- |
| **性能要求**  | 筛选、搜索、模式切换后前端需快速重渲染；导出任务需在可接受时延内完成。 | 常规数据量（≤1,000 行）下：筛选/搜索结果刷新 ≤ 800ms；模式切换 ≤ 1.2s；导出触发后 3s 内开始下载。               |
| **容量要求**  | 列表与详情需支持批量事件展示与多条件筛选叠加，不出现明显卡顿。     | 在 5,000 行数据集下可完成时间+状态+关键词+表头过滤组合操作；页面无白屏，交互可连续执行。                            |
| **数据一致性** | 状态、均价、日期、触发来源在列表、详情、导出文件中保持统一口径。    | 抽样核对 20 条记录：页面与导出文件关键字段一致率 100%；状态映射与 Avg.Price 计算规则一致。                      |
| **可用性**   | 过滤菜单、空状态、确认弹窗需给出清晰反馈，避免误操作。         | 过滤菜单可点击外部自动关闭；无数据展示 No Event；导出操作均出现 Confirm 弹窗且可取消。                         |
| **兼容性**   | 导出文件需兼容主流办公软件，文档在常见浏览器环境可正常渲染。      | `.xls` 文件可被 Excel/WPS 正常打开；Chrome 最新稳定版下表格与 HTML 列表渲染正常。                     |
| **可追溯性**  | 导出文件命名与筛选上下文应支持复盘追踪。                | 历史模式导出文件名包含日期标签；可根据文件名与导出时间定位对应筛选区间。                                         |
| **当前约束**  | 现阶段部分字段仍为占位展示，后续需接入真实数据映射。          | DER Events 的 Pricing Region、VPP 字段当前为 `-`；VPP 列表 Trigger 字段当前为 `-`，纳入后续版本迭代。 |
