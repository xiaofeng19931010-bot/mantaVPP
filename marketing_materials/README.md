# Manta PV ESS Marketing Package

This package contains professional marketing materials for the Manta PV ESS (Photovoltaic Energy Storage System) product.

## Contents

1.  **PV_ESS_Marketing_Strategy.docx**
    *   **Format**: Microsoft Word
    *   **Content**: Detailed product analysis, market strategy, user personas, and competitive landscape.
    *   **Usage**: Use this as the source of truth for detailed proposals, white papers, or website copy.

2.  **PV_ESS_Marketing_Presentation.pptx**
    *   **Format**: Microsoft PowerPoint
    *   **Content**: 15+ slide presentation covering the company, problem, solution, features, and roadmap.
    *   **Usage**: Ready for investor pitches or customer demos.
    *   **Design**: Uses Manta Web brand colors (Primary Green: #2E9F58).

3.  **Source Scripts** (`create_marketing_doc.js`, `create_marketing_ppt.js`)
    *   Node.js scripts used to generate the files.
    *   You can modify the content in these scripts and run `node create_marketing_doc.js` or `node create_marketing_ppt.js` to regenerate the documents.

## Customization Guide

### Adding Visuals
The documents contain placeholders for images (e.g., `[Dashboard Screenshot Placeholder]`).
*   **Word**: Click the placeholder text and insert your high-resolution screenshots.
*   **PowerPoint**: Delete the gray placeholder boxes and paste your screenshots.

### Editing Content
Both files are native Office formats. You can open them directly in Microsoft Word or PowerPoint to edit text, change colors, or add new sections.

### Branding
The materials follow the Manta Design System:
*   **Primary Color**: #2E9F58
*   **Accent Color**: #3EC064
*   **Font**: Arial / Calibri (Standard web-safe fonts used for compatibility).
